## Praktikum 12 - VueJS Components dan Routing (SPA)
Pada praktikum ini dilakukan pengembangan aplikasi Frontend API yang telah dibuat sebelumnya dengan menerapkan konsep Vue Components dan Vue Router. Dengan Vue Router, aplikasi dapat berpindah halaman tanpa melakukan refresh browser (Single Page Application/SPA).

## Tujuan Praktikum

1. Memahami konsep Vue Components.
2. Memahami konsep Client-Side Routing menggunakan Vue Router.
3. Mengimplementasikan Single Page Application (SPA) menggunakan VueJS dan Vue Router.


## Software yang Digunakan

- Visual Studio Code
- XAMPP
- VueJS 3
- Vue Router 4
- Axios
- CodeIgniter 4 (REST API)
- Google Chrome

## Hasil Akhir Program

Fitur yang berhasil dibuat yaitu:

- Single Page Application (SPA).
- Vue Components.
- Vue Router.
- CRUD Artikel.
- REST API menggunakan Axios.
- Halaman About.
- Navigasi antar halaman tanpa refresh browser.

## Kesimpulan

Pada praktikum ini berhasil dibuat aplikasi Single Page Application (SPA) menggunakan VueJS Components dan Vue Router. Dengan penerapan Vue Router, perpindahan halaman dapat dilakukan tanpa hard reload browser sehingga memberikan pengalaman pengguna yang lebih cepat dan interaktif. Selain itu, integrasi dengan REST API menggunakan Axios memungkinkan proses pengelolaan data artikel dilakukan secara dinamis dan efisien.
